# Algorithms

用 Python 学习和实现经典算法与数据结构。

## 目录

| 目录 | 主题 |
|------|------|
| `01_sorting/` | 排序算法 |
| `02_searching/` | 搜索算法 |
| `03_data_structures/` | 数据结构 |
| `04_dynamic_programming/` | 动态规划 |
| `05_greedy/` | 贪心算法 |
| `06_backtracking/` | 回溯算法 |
| `07_graph/` | 图算法 |
| `utils/` | 工具函数 |

## 快速开始

```bash
pip install -r requirements.txt
pytest algorithms/ -v
```
